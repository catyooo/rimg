随机图片API | 由Catyo工作室API项目组(api.catyo.cn)责任运营

图片默认均为https，采用新浪图片，高速访问

二次元随机图 API基本调用格式：

https://api.catyo.cn/rimg/2dyrimg.php

Catyo精选随机图 API基本调用格式：

https://api.catyo.cn/rimg/catyorimg.php

参数：

return=json/http/https/img

JSON调用格式

https://api.catyo.cn/rimg/2dyrimg.php?return=json

JSON数据

{
    "code":"200" #图片状态码
    "acgurl":"https:\/\/tva1.sinaimg.cn\/large\/0072Vf1pgy1foxk7n4ffij31hc0u0wyt.jpg" #图片地址
    "width":"2048" #图片宽
    "height":"1152" #图片高
}
